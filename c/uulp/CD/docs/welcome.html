Welcome, Unix Teacher

Thank you for your interest in using Understanding Unix/Linux Programming for your course. I prepared this CD to make it easy for you to teach a course based on the book. This CD includes everything you need to teach a course: lecture outlines, classroom handouts, homework projects, final exams, and lots of graphics. The CD also includes source code for all the programs in the book as well as the sources for the images in the book.

In outline, this CD contains:

  1. Classroom material
    • Lecture Outlines
    • Sample programs for class
    • Listings of class sample programs
    • What to write on the board/Slideshow
    • Thumbnail pictures of slideshow
  2. Solutions to programming exercises
  3. All the code in the book (with Makefiles)
  4. 12 Final exams
  5. 12 Programming projects, many with test scripts
  6. Binary choice questions for each chapter
  7. Full color images of the pictures in the book
  8. 3D Models for the pictures in the book
  9. Guides to using the material

I have used and refined these materials in my course for twelve years. You can simply print out pages from the CD and give lectures. And grade papers, too, I guess.

Most of the material is grouped by chapter. For each chapter you will find classroom handouts, source code, solutions to exercises, binary choice tests, and graphics. Material not tied to specific chapters -- programming projects and final exams -- are in their own directories.

I hope you find this CD useful and enjoy using the book for your course. Please feel free to send me email.
