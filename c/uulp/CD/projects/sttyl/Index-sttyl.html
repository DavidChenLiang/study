  • The `real' thing

    Please note that there is no `correct' version on this machine. There is a program called /bin/stty and a version called /usr/local/bin/stty. The first one is from DEC and the second one is from FSF. They do the same thing, more or less, but their output and error messages differ. Therefore, there is no `correct' output you need to worry about.
