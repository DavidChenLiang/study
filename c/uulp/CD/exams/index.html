What
This section contains 12 complete final exams. These are exams I used in my course. Students had two hours for the exam, and most of these exams were about the right length for that time. These are numbered in chronological order being given in May 1991 through May 2002.

These are not secret; copies are in on file in the Harvard Extension School library. On the other hand, any student who goes to the trouble to get copies of all these exams and figure out answers to all the questions will probably learn a lot.

Design
All the exams test factual knowledge, basic skills used in one or two of the programming projects, and understanding of Unix programming design.

There are many good answers to most of these questions, particularly the programming and design questions. The number of points per question is pretty low, so it is not hard to make up fair grading even for a variety of answers.

You are of course free to cut, paste, edit, delete these questions in any way you like.

File Format
Since this is a Unix course, the exams are coded with troff markup tags with some sections designed as tables using the tbl table preprocessor language. The script called MakeExam takes the name of a troff file and writes pdf output to standard output. Therefore, 
	MakeExam finalXX.tr > final.pdf
will work. If you want, instead, to create Postscript, do this: 
	tbl finalXX.tr | groff -Tps > finalXX.ps
The script called MakeExams takes the names of one or more exam files on the command line and creates pdf files with corresponding names. For example: 
	MakeExams *.tr
will generate pdf versions of all the exams.
